# Red-Black Tree
Библиотека на C++17 для работы с красно-чёрным деревом с поддержкой циклического буфера, включая добавление, удаление и сохранение/загрузку из файла.

```bash
mkdir build
cd build
cmake ..
cmake --build .
./test_tree  


cmake .. -G "MinGW Makefiles"
mingw32-make
./test_tree   # запуск тестов вручную
```
